# Poker

## Vercel Deployment

To deploy this project on Vercel, use the following GitHub repository link:

**https://github.com/Aymenhajbi/Poker**

### Steps to deploy on Vercel:
1. Go to [vercel.com](https://vercel.com) and sign in.
2. Click **"Add New Project"**.
3. Import the repository using the link above: `https://github.com/Aymenhajbi/Poker`
4. Follow the setup prompts and click **Deploy**.
